Exercice 1

Dans cet exercice, nous avons utilisé l’inspecteur de code pour repérer et comprendre les différentes zones qui structurent la page web. Objectif : identifier clairement le zoning du site.

🧱 Zoning de la page analysée

  • <header> — En-tête du site (titre)
  • <nav> — Menu de navigation principal
  • <aside> — Menu latéral gauche
  • <article> — Zone de contenu principal
  • <footer> — Pied de page

📂 Contenu de l'article :

  • Titres : <h1>, <h2>, <h3>, <h4>, <h5>
  • Paragraphes : <p>
  • Listes : <ul>, <li>
  • Liens : <a>
  • Images : <img>
